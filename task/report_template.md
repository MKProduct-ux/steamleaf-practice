В рамках практики выполняется работа над проектом STEAMLEAF. STEAMLEAF — это разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Дополнительно в рамках практики создаются сайт проекта и Telegram-бот, которые помогают представить игру, её мир, персонажей, визуальные материалы и основные игровые механики.

## 1. Общая информация о проекте

В данном разделе указывается основная информация о проекте.

### Название проекта

STEAMLEAF.

### Краткое описание проекта

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Игрок управляет молодым промоутером, который раздаёт листовки в стимпанк-городе, взаимодействует с персонажами, выполняет задания и влияет на социальный рейтинг.

Проект сочетает:

- стимпанк-эстетику;
- социальную сатиру;
- квестовую структуру;
- систему социального рейтинга;
- раздачу листовок как часть геймплея.

### Цель проекта

Цель проекта — разработка игры STEAMLEAF и сопутствующих материалов для её презентации.

В рамках проектной практики создаются:

- сайт проекта;
- Telegram-бот;
- документация;
- итоговые отчётные материалы.

### Задачи проекта

В рамках проекта решаются следующие задачи:

- разработка игрового мира;
- создание персонажей;
- подготовка визуальных материалов;
- разработка квестов и игровых механик;
- создание сайта проекта;
- создание Telegram-бота;
- оформление документации;
- подготовка итогового отчёта.

### Команда проекта

Над проектом STEAMLEAF работает команда студентов, распределённая по рабочим направлениям.

В проекте участвуют следующие направления:

- сценаристы;
- гейм-дизайнеры;
- программисты;
- 2D-художники;
- 3D-художники;
- графические дизайнеры;
- звукорежиссёр;
- менеджеры по продвижению;
- разработчики технической документации.

## 4. Описание достигнутых результатов по проектной практике

В данном разделе описываются результаты, полученные в ходе практики.

### Создание репозитория

Был создан Git-репозиторий `steamleaf-practice`.

В репозитории подготовлена структура:

```text
/
├── README.md
├── docs/
├── reports/
├── site/
├── src/
├── task/
└── .gitignore
```

### Документация

В папке `docs/` подготовлена документация по проекту.

Документация включает:

- описание хода выполнения практики;
- концепцию Telegram-бота;
- описание сайта;
- описание материалов проекта STEAMLEAF;
- описание команды проекта.

### Сайт

В папке `site/` размещается статический сайт проекта.

Сайт содержит информацию об игре STEAMLEAF, её мире, персонажах, команде проекта, листовках, ходе разработки и Telegram-боте.

Сайт разрабатывается с использованием HTML и CSS.

### Telegram-бот

В папке `src/` размещается исходный код Telegram-бота.

Бот создаётся на Python и предназначен для интерактивного знакомства пользователя с игрой STEAMLEAF.

Планируемый бот включает:

- команду `/start`;
- главное меню;
- раздел «Об игре»;
- раздел «Персонажи»;
- раздел «Районы»;
- раздел «Листовки»;
- мини-игру «Рабочий день промоутера»;
- отображение социального рейтинга SR;
- ссылку на сайт проекта.

### Отчётные материалы

В папке `reports/` размещаются итоговые отчётные материалы:

- отчёт в формате DOCX;
- отчёт в формате PDF.

### Текущий результат

На текущем этапе подготовлены:

- структура Git-репозитория;
- документация в формате Markdown;
- описание сайта;
- концепция Telegram-бота;
- описание команды проекта;
- папки для размещения сайта, исходного кода и отчётов.

После завершения разработки будут добавлены:

- полный код сайта;
- полный код Telegram-бота;
- итоговый отчёт в формате DOCX;
- итоговый отчёт в формате PDF.
