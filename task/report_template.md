## 1. Общая информация о проекте

В данном разделе указывается основная информация о проекте.

### Название проекта

STEAMLEAF.

### Краткое описание проекта

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Игрок управляет молодым промоутером, который раздаёт листовки в стимпанк-городе, взаимодействует с персонажами, выполняет задания и влияет на социальный рейтинг.

### Цель проекта

Создание игры STEAMLEAF, а также сопутствующих материалов для её презентации: сайта, Telegram-бота и документации.

### Задачи проекта

В рамках проекта решаются следующие задачи:

- разработка игрового мира;
- создание персонажей;
- подготовка визуальных материалов;
- разработка квестов и игровых механик;
- создание сайта проекта;
- создание Telegram-бота;
- оформление документации.

## 2. Общая характеристика деятельности организации или заказчика проекта

В данном разделе описывается организация или заказчик проекта.

### Наименование заказчика

Проект выполняется в рамках проектной деятельности и проектной практики.

### Организационная структура

Работа выполняется участниками проекта под руководством куратора проектной деятельности и ответственного по проектной практике.

### Описание деятельности

Деятельность связана с разработкой цифрового проекта — игры STEAMLEAF и сопутствующих материалов.

В рамках практики особое внимание уделяется:

- оформлению Git-репозитория;
- созданию сайта;
- разработке Telegram-бота;
- подготовке документации;
- подготовке итогового отчёта.

## 3. Описание задания по проектной практике

В данном разделе описывается задание, выполненное в рамках проектной практики.

### Базовая часть

Базовая часть включает:

- создание Git-репозитория;
- оформление документации в формате Markdown;
- разработку статического сайта;
- подготовку отчётных материалов.

### Вариативная часть

Вариативная часть связана с разработкой Telegram-бота для игры STEAMLEAF.

Telegram-бот должен знакомить пользователя с проектом и выполнять функции интерактивного дополнения к игре.

Планируемые функции Telegram-бота:

- приветствие пользователя;
- главное меню;
- описание игры;
- раздел персонажей;
- раздел районов;
- раздел листовок;
- мини-игра «Рабочий день промоутера»;
- отображение социального рейтинга SR;
- ссылка на сайт проекта.
