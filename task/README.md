# Задание на проектную практику

## Общая информация

Проектная практика выполняется в рамках учебной деятельности и направлена на закрепление навыков работы с Git, Markdown, HTML, CSS и инструментами разработки программных проектов.

В рамках практики создаётся Git-репозиторий, оформляется документация, разрабатывается статический сайт и выполняется вариативная часть задания.

## Проект

**Наименование проекта:** STEAMLEAF

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Игра рассказывает о молодом промоутере, который раздаёт листовки в стимпанк-городе, взаимодействует с персонажами, выполняет задания и влияет на социальный рейтинг.

## Цель проектной практики

Цель проектной практики — создать информационный сайт и Telegram-бота для проекта STEAMLEAF, а также оформить результаты работы в Git-репозитории.

## Основные задачи

В рамках практики необходимо выполнить следующие задачи:

- создать Git-репозиторий;
- подготовить структуру репозитория;
- оформить документацию в формате Markdown;
- создать статический сайт проекта;
- разработать Telegram-бота;
- подготовить итоговый отчёт;
- разместить результаты работы в репозитории.

## Базовая часть задания

Базовая часть включает:

- настройку Git-репозитория;
- создание документации;
- разработку статического сайта;
- оформление материалов проекта;
- подготовку отчётных документов.

## Вариативная часть задания

Вариативная часть задания связана с разработкой Telegram-бота для игры STEAMLEAF.

## Формулировка вариативной части

Разработать Telegram-бота для проекта STEAMLEAF, который будет знакомить пользователя с игрой, её миром, персонажами, районами, листовками и системой социального рейтинга.

## Назначение Telegram-бота

Telegram-бот STEAMLEAF создаётся как интерактивное дополнение к игре.

Бот должен выполнять следующие функции:

- приветствовать пользователя;
- показывать главное меню;
- рассказывать об игре;
- показывать информацию о персонажах;
- рассказывать о районах города;
- показывать листовки из мира игры;
- запускать мини-игру «Рабочий день промоутера»;
- отображать социальный рейтинг SR;
- давать ссылку на сайт проекта.

## Ожидаемый результат

По итогам выполнения практики должны быть подготовлены:

- Git-репозиторий с корректной структурой;
- документация в формате Markdown;
- сайт проекта STEAMLEAF;
- Telegram-бот;
- итоговый отчёт в формате DOCX;
- итоговый отчёт в формате PDF.

## Используемые технологии

В проекте используются:

- Git;
- GitHub;
- Markdown;
- HTML;
- CSS;
- Python;
- Telegram Bot API.

## Структура репозитория

Репозиторий содержит следующие основные папки:

- `docs/` — документация;
- `reports/` — отчётные материалы;
- `site/` — статический сайт;
- `src/` — исходный код Telegram-бота;
- `task/` — текст задания и шаблон отчёта.

## Итог

Данное задание объединяет разработку сайта, создание Telegram-бота и оформление документации по проекту STEAMLEAF.
