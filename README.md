## Описание проекта

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Игрок управляет молодым промоутером, который раздаёт листовки в стимпанк-городе, взаимодействует с персонажами, выполняет задания и влияет на социальный рейтинг.

Проект сочетает:

- стимпанк-эстетику;
- социальную сатиру;
- квестовую структуру;
- систему социального рейтинга SR;
- раздачу листовок как часть геймплея;
- визуальный стиль, основанный на бумажных интерфейсах, листовках и индустриальной атмосфере.

## Цель проектной практики

Цель проектной практики — создать информационный сайт и Telegram-бота для проекта STEAMLEAF, а также оформить результаты работы в Git-репозитории.

## Задачи проектной практики

В рамках практики необходимо:

- создать Git-репозиторий;
- подготовить структуру репозитория;
- оформить документацию в формате Markdown;
- разработать статический сайт проекта;
- разработать Telegram-бота;
- подготовить итоговый отчёт;
- разместить результаты работы в репозитории.

## Вариативная часть задания

Вариативная часть проектной практики связана с разработкой Telegram-бота для игры STEAMLEAF.

### Формулировка вариативной части

Разработать Telegram-бота для проекта STEAMLEAF, который будет знакомить пользователя с игрой, её миром, персонажами, районами, листовками и системой социального рейтинга.

## Сайт проекта

Сайт проекта размещается в папке `site/`.

Сайт создаётся как информационная витрина игры STEAMLEAF.

Планируемые разделы сайта:

- главная страница;
- о проекте;
- мир игры;
- персонажи;
- команда проекта;
- листовки;
- журнал разработки;
- Telegram-бот;
- ресурсы.

## Telegram-бот

Исходный код Telegram-бота размещается в папке `src/`.

Telegram-бот STEAMLEAF создаётся как интерактивное дополнение к игре.

Бот задуман как «терминал промоутера», через который пользователь может познакомиться с миром игры, персонажами, районами, листовками и системой социального рейтинга.

Планируемые функции бота:

- запуск по команде `/start`;
- приветственное сообщение;
- главное меню;
- раздел «Об игре»;
- раздел «Персонажи»;
- раздел «Районы»;
- раздел «Листовки»;
- мини-игра «Рабочий день промоутера»;
- отображение социального рейтинга SR;
- ссылка на сайт проекта.

## Описание папок

### `docs/`

Папка содержит документацию по проектной практике.

В ней размещаются:

- описание хода выполнения практики;
- концепция Telegram-бота;
- описание сайта;
- описание материалов проекта;
- информация о команде проекта.

### `reports/`

Папка предназначена для итоговых отчётных материалов.

После завершения работы в неё будут добавлены:

- отчёт в формате DOCX;
- отчёт в формате PDF.

### `site/`

Папка содержит файлы статического сайта проекта.

Основные файлы:

- `index.html` — главная страница сайта;
- `css/style.css` — стили сайта;
- `images/` — изображения, логотипы, скриншоты, листовки и другие визуальные материалы.

### `src/`

Папка содержит исходный код Telegram-бота.

На текущем этапе здесь размещается базовая структура, которая позже будет дополнена полноценным кодом бота.

### `task/`

Папка содержит текст задания и шаблон отчёта по проектной практике.
