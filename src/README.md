# Исходный код Telegram-бота STEAMLEAF

В этой папке размещается исходный код Telegram-бота STEAMLEAF, создаваемого в рамках вариативной части проектной практики.

## Назначение папки

Папка src/ предназначена для хранения программного кода проекта.

В рамках данной практики здесь будет размещён код Telegram-бота, который выступает интерактивным дополнением к игре STEAMLEAF.

## Назначение Telegram-бота

Telegram-бот STEAMLEAF создаётся как бот-промоутер игры.

Он помогает пользователю быстро познакомиться с проектом, его атмосферой, персонажами, районами, листовками и системой социального рейтинга SR.

Бот задуман не как обычная справка, а как условный «промоутерский терминал» из мира STEAMLEAF.

## Основная идея бота

Пользователь взаимодействует с ботом так, будто подключился к терминалу из игрового мира.

Через меню он может узнать:

- о чём игра STEAMLEAF;
- кто главный герой;
- какие персонажи есть в игре;
- какие районы существуют в городе;
- какие листовки встречаются в мире игры;
- что такое социальный рейтинг SR;
- где находится сайт проекта.

Дополнительно бот содержит интерактивные элементы:

- случайную листовку дня;
- викторину по листовкам;
- случайные предсказания от Лилит.

## Основные функции бота

Планируемые функции Telegram-бота:

- запуск по команде /start;
- вывод приветственного сообщения;
- отображение главного меню;
- раздел «О чём наша игра»;
- раздел «Главный герой»;
- раздел «Персонажи»;
- раздел «Районы»;
- раздел «Листовки»;
- раздел «Что такое SR»;
- раздел «Листовка дня»;
- раздел «Викторина»;
- раздел «Предсказание Лилит»;
- раздел со ссылкой на сайт игры;
- команда /help с краткой справкой по использованию бота.
