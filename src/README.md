# Исходный код Telegram-бота STEAMLEAF

В этой папке размещён исходный код Telegram-бота проекта STEAMLEAF.

## Основной файл

- `main.py` — основной файл Telegram-бота.

## Назначение бота

Telegram-бот STEAMLEAF создан как интерактивное дополнение к сайту и игре.

Бот выполняет роль терминала промоутера и позволяет пользователю:

- узнать, о чём игра STEAMLEAF;
- познакомиться с главным героем;
- посмотреть персонажей;
- узнать о районах города;
- изучить листовки;
- узнать, что такое социальный рейтинг SR;
- получить случайную листовку дня;
- пройти викторину по листовкам;
- получить предсказание Лилит;
- перейти на сайт проекта.

## Основные функции

В боте реализованы:

- команда `/start`;
- команда `/help`;
- главное меню;
- раздел «О чём наша игра»;
- раздел «Главный герой»;
- раздел «Персонажи»;
- раздел «Районы»;
- раздел «Листовки»;
- раздел «Что такое SR»;
- случайная «Листовка дня»;
- викторина по листовкам;
- предсказания Лилит;
- кнопка перехода на сайт проекта.

## Используемые технологии

- Python
- Aiogram
- Telegram Bot API
- python-dotenv

## Переменные окружения

Для запуска бота используется файл `.env`.

Пример файла:

```env
BOT_TOKEN=your_telegram_bot_token_here
SITE_URL=https://mkproduct-ux.github.io/steamleaf-practice/
```

Настоящий файл `.env` не размещается в репозитории, потому что содержит секретный токен Telegram-бота.

В репозитории размещается только файл `.env.example`, где показан пример необходимых переменных.

## Установка зависимостей

Перед запуском необходимо установить зависимости из файла `requirements.txt`.

```bash
pip install -r requirements.txt
```

## Запуск бота

После установки зависимостей и настройки файла `.env` бот запускается командой:

```bash
python src/main.py
```

## Изображения для бота

Бот использует изображения персонажей, районов и листовок.

Изображения размещаются в папке `assets/`:

```text
assets/
├── characters/
├── districts/
└── leaflets/
```
