# Отчёты по проектной практике

В этой папке размещаются отчётные материалы по проектной практике.

## Назначение папки

Папка `reports/` предназначена для хранения итогового отчёта по проектной практике и его PDF-версии.

Отчёт оформляется на основе официального шаблона отчёта по проектной практике.

## Планируемые файлы

В папке будут размещены:

- `practice_report_template.docx` — официальный шаблон отчёта;
- `report.docx` — итоговый отчёт в формате DOCX;
- `report.pdf` — итоговый отчёт в формате PDF.

## Структура итогового отчёта

Итоговый отчёт включает следующие разделы:

- введение;
- общая информация о проекте;
- общая характеристика деятельности организации или заказчика проекта;
- описание задания по проектной практике;
- описание достигнутых результатов по проектной практике;
- заключение;
- список использованной литературы;
- приложения.

## Содержание отчёта

В итоговом отчёте будут описаны:

- проект STEAMLEAF;
- цель и задачи проектной практики;
- структура Git-репозитория;
- разработка сайта проекта;
- разработка Telegram-бота;
- команда проекта;
- использованные технологии;
- достигнутые результаты.

## Статус

На текущем этапе папка подготовлена для размещения отчётных материалов.

Итоговый отчёт будет добавлен после завершения разработки сайта и Telegram-бота.
