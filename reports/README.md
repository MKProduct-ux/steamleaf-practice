# Отчёты по проектной практике

В этой папке размещаются итоговые отчётные материалы по проектной практике.

## Назначение папки

Папка `reports/` предназначена для хранения готового отчёта по практике в форматах DOCX и PDF.

Отчёт будет подготовлен после завершения основных этапов работы:

- создания структуры Git-репозитория;
- оформления документации;
- разработки сайта проекта;
- разработки Telegram-бота;
- тестирования результата;
- описания выполненной работы.

## Планируемые файлы

После завершения работы в папку будут добавлены следующие файлы:

- `report.docx` — итоговый отчёт в формате DOCX;
- `report.pdf` — итоговый отчёт в формате PDF.

## Содержание отчёта

Итоговый отчёт должен включать:

- введение;
- цель и задачи практики;
- описание проекта STEAMLEAF;
- описание структуры репозитория;
- описание разработки сайта;
- описание разработки Telegram-бота;
- описание использованных технологий;
- результаты работы;
- вывод.

## Статус

На текущем этапе папка подготовлена для дальнейшего размещения отчётных материалов.
