# Команда проекта STEAMLEAF

## Общая информация

Над проектом STEAMLEAF работает большая команда студентов, разделённая по рабочим направлениям.

В проект входят следующие рабочие группы:

- сценаристы;
- гейм-дизайнеры;
- программисты;
- 2D-художники;
- 3D-художники;
- графические дизайнеры;
- звукорежиссёр;
- менеджеры по продвижению;
- разработчики технической документации.

## Роль команды в проекте

Команда STEAMLEAF занимается разработкой игры, её визуального стиля, сценария, игровых механик, технической документации и продвижения.

В рамках проектной практики отдельное внимание уделяется созданию сайта и Telegram-бота, которые помогают представить проект и сделать его более доступным для пользователей.
