# Материалы проекта STEAMLEAF

## Описание

В рамках проекта STEAMLEAF используются визуальные, текстовые и интерактивные материалы, которые помогают представить игру, её мир, персонажей и атмосферу.

Материалы размещены на сайте проекта и в документации репозитория.

## Основные материалы

К материалам проекта относятся:

- логотип проекта;
- изображение города STEAMLEAF;
- изображения районов города;
- изображения персонажей;
- изображения листовок;
- описание мира игры;
- описание персонажей;
- описание команды;
- журнал разработки;
- VK Видео;
- Telegram-бот.

## Изображения

На сайт добавлены изображения следующих типов:

### Логотип

Логотип используется в шапке сайта и помогает визуально обозначить проект.

### Город STEAMLEAF

Изображение города используется на главном экране сайта.

### Районы города

Для раздела «Мир игры» используются изображения районов:

- трущобы;
- средний район;
- богатый район;
- элитный район.

### Персонажи

Для раздела «Персонажи» используются изображения героев и жителей мира STEAMLEAF.

### Листовки

Листовки являются важной частью визуального стиля проекта.  
Они связаны с геймплеем, рекламной культурой города и сатирой.
