# Документация проекта STEAMLEAF

В этой папке размещается документация по проектной практике.

Документация оформлена в формате Markdown и описывает процесс выполнения практики, разработку сайта и создание Telegram-бота для игры STEAMLEAF.

## Содержание папки

- `practice_documentation.md` — описание хода выполнения проектной практики.
- `bot_concept.md` — концепция Telegram-бота STEAMLEAF.
- `site_description.md` — описание сайта проекта.
- `project_materials.md` — описание материалов игры STEAMLEAF.
- `team.md` — информация о команде проекта STEAMLEAF.

## Назначение документации

Документация нужна для фиксации этапов работы, описания принятых решений и подготовки итогового отчёта по проектной практике.

В документах описываются:

- цель проекта;
- задачи проектной практики;
- структура репозитория;
- материалы игры STEAMLEAF;
- разработка сайта;
- разработка Telegram-бота;
- планируемые результаты работы.

## Проект

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

В рамках проектной практики создаются:

- статический сайт о проекте;
- Telegram-бот по игре;
- документация;
- итоговый отчёт.
