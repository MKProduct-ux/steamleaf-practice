# Описание сайта STEAMLEAF

## Назначение сайта

Сайт STEAMLEAF создан в рамках проектной практики для представления игры, её мира, персонажей, визуальных материалов, команды и Telegram-бота.

Сайт опубликован через GitHub Pages.

## Ссылка на сайт

https://mkproduct-ux.github.io/steamleaf-practice/

## Структура сайта

Сайт включает следующие разделы:

1. Главный экран.
2. О проекте.
3. Мир игры.
4. Персонажи.
5. Команда.
6. Листовки.
7. Журнал разработки.
8. Видео проекта.
9. Telegram-бот.
10. Ресурсы.

## Главный экран

На главном экране размещены:

- название проекта STEAMLEAF;
- краткое описание игры;
- изображение города STEAMLEAF;
- кнопки перехода к разделам сайта;
- краткие показатели проекта.

## Раздел «О проекте»

В разделе описывается основная идея игры:

- жанр;
- главный герой;
- стимпанк-город;
- социальный рейтинг SR;
- роль листовок;
- сатирическая направленность проекта.

## Раздел «Мир игры»

В этом разделе представлены районы города:

- трущобы;
- средний район;
- богатый район;
- элитный район.

Для каждого района добавлено изображение и краткое описание.

## Раздел «Персонажи»

В разделе представлены персонажи STEAMLEAF.  
Для каждого персонажа добавлены изображение, имя и краткое описание.

## Раздел «Команда»

Раздел содержит информацию о команде проекта и распределении ролей.

## Раздел «Листовки»

В разделе представлены листовки как важная часть визуального стиля, геймплея и юмора проекта.

## Раздел «Журнал разработки»

В журнале разработки отражён ход работы над игрой и ход выполнения проектной практики.

В разделе отдельно указано, что над проектом в рамках проектной практики работают:

- Лагутенко Алеся;
- Кинёв Максим.
