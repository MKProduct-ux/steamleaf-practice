## Описание проекта

STEAMLEAF — разрабатываемая сюжетная игра в жанре нарративного квеста с элементами RPG.

Действие игры происходит в стимпанк-городе, где социальный рейтинг SR влияет на доступ к районам, отношение персонажей и игровые возможности. Главный герой — молодой промоутер, который раздаёт листовки, выполняет задания и взаимодействует с жителями города.

Проект сочетает стимпанк-эстетику, социальную сатиру, квестовую структуру, систему социального рейтинга и визуальный стиль, основанный на бумажных интерфейсах, листовках и индустриальной атмосфере.

## Цель проектной практики

Целью проектной практики является создание цифрового представления проекта STEAMLEAF, включающего сайт, Telegram-бота, документацию и материалы, демонстрирующие концепцию игры.

## Задачи проектной практики

В рамках практики были поставлены и выполнены следующие задачи:

1. Создать GitHub-репозиторий проекта.
2. Подготовить структуру репозитория.
3. Разработать статический сайт проекта.
4. Подготовить документацию в формате Markdown.
5. Добавить описание игры, мира, персонажей, команды и листовок.
6. Подключить изображения и визуальные материалы проекта.
7. Добавить видео проекта на сайт.
8. Создать Telegram-бота.
9. Добавить рабочие ссылки на сайт, GitHub, Telegram-бота, Telegram-канал, VK и VK Видео.
10. Подготовить материалы для итогового отчёта по практике.

## Ход выполнения работы

### 1. Создание репозитория

На первом этапе был создан GitHub-репозиторий проекта.

В репозитории была подготовлена структура папок и файлов, соответствующая требованиям проектной практики.

Были созданы основные папки:

- `docs` — для документации;
- `reports` — для отчётных материалов;
- `site` — для сайта проекта;
- `src` — для исходного кода Telegram-бота;
- `task` — для задания и шаблонов отчёта.

### 2. Подготовка документации

В папке `docs` были подготовлены Markdown-файлы с описанием проекта:

- `practice_documentation.md` — ход выполнения проектной практики;
- `site_description.md` — описание сайта проекта;
- `bot_concept.md` — описание Telegram-бота;
- `project_materials.md` — описание материалов проекта;
- `team.md` — информация о команде проекта.

Документация была обновлена с учётом финального состояния проекта.

### 3. Разработка сайта

Был разработан статический сайт проекта STEAMLEAF.

Сайт размещён в папке `site/`.

- `site/css/style.css` — файл стилей;
- `site/images/` — изображения проекта;
- `site/fonts/` — шрифты сайта.

Сайт опубликован с помощью GitHub Pages.
