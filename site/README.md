# Сайт проекта STEAMLEAF

В этой папке размещается статический сайт проекта STEAMLEAF.

Сайт создаётся в рамках проектной практики и предназначен для презентации игры, её мира, персонажей, визуального стиля, листовок и Telegram-бота.

## Назначение сайта

Сайт является информационной витриной проекта STEAMLEAF.

Он должен познакомить пользователя с основной идеей игры и показать материалы, подготовленные в рамках проектной деятельности.
