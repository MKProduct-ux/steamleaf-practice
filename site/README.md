# Сайт проекта STEAMLEAF

В этой папке размещается статический сайт проекта STEAMLEAF.

Сайт создаётся в рамках проектной практики и предназначен для презентации игры, её мира, персонажей, визуального стиля, команды разработчиков, листовок и Telegram-бота.

## Назначение сайта

Сайт является информационной витриной проекта STEAMLEAF.

Он должен познакомить пользователя с основной идеей игры, показать материалы проекта, представить команду разработчиков и продемонстрировать дополнительные материалы, созданные в рамках проектной практики.

## Структура папки

```text
site/
├── README.md
├── index.html
├── css/
│   └── style.css
└── images/
    └── .gitkeep
```

## Основные файлы

- `index.html` — главная страница сайта.
- `css/style.css` — файл стилей сайта.
- `images/` — папка для изображений, логотипов, скриншотов, листовок и других визуальных материалов.

## Планируемые разделы сайта

Сайт будет включать следующие разделы:

- главная страница;
- о проекте;
- мир игры;
- персонажи;
- команда проекта;
- листовки;
- журнал разработки;
- Telegram-бот;
- ресурсы.

## Главная страница

На главной странице будет размещено название игры STEAMLEAF, краткое описание проекта и переходы к основным разделам сайта.

Главная страница должна сразу передавать атмосферу проекта: стимпанк-город, социальный рейтинг, листовки, персонажи и сатирический тон игры.

## Раздел «О проекте»

Раздел будет содержать описание игры, жанра и основной идеи.

STEAMLEAF — сюжетная игра в жанре нарративного квеста с элементами RPG. Игрок управляет промоутером, который раздаёт листовки, выполняет задания и взаимодействует с персонажами стимпанк-города.

## Раздел «Мир игры»

Раздел будет посвящён городу, системе социального рейтинга SR и районам.

Планируемые районы:

- Трущобы;
- Средний район;
- Богатый район;
- Элитный район.

Социальный рейтинг влияет на доступ к районам, отношение персонажей и возможности главного героя.

## Раздел «Персонажи»

В разделе будут представлены основные персонажи игры:

- главный герой;
- Графиня Мокси;
- Рэй;
- Лилит;
- SP-67;
- Валера;
- босс;
- доктор Зуд;

Для каждого персонажа планируется указать краткое описание, роль в мире игры и визуальные материалы.
